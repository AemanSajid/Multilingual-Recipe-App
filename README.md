BiteBook is a Flutter-based recipe application designed to help users discover, search, save, and explore recipes in a clean and interactive interface.

Core Features
1. Home Screen

Displays Recipe of the Day to highlight featured content.

Includes a centered search bar for quick recipe lookup.

Shows category chips (e.g., Breakfast, Dessert, Fast Food) for easy filtering.

Uses a grid layout with small recipe cards (5 per row) for compact browsing.

2. Recipe Cards

Each card displays:

Recipe image

Recipe title

Like count

Includes bookmark button to save recipes.

Smooth hover and tap animations improve user interaction.

Arrow navigation allows moving to detailed recipe view.

3. Search & Filtering

Users can:

Search recipes by name

Filter recipes by category

Results update dynamically for better usability.

4. Saved Recipes

Bookmarked recipes are stored locally using SharedPreferences.

Ensures saved recipes remain available even after app restart.

Users can easily access their favorite recipes anytime.

5. UI & UX Design

Clean, modern, and minimal design.

Consistent spacing, icons, and color usage.

Responsive layout for different screen sizes.

Focus on accessibility and ease of navigation.

6. Footer Section

Includes social media links.

Adds branding and completeness to the app layout.

Technical Stack

Frontend: Flutter

State Management: Local state

Local Storage: SharedPreferences

Architecture: Modular and scalable UI components

Purpose & Benefits

Makes recipe discovery fast and enjoyable.

Helps users organize favorite recipes.

Offers a visually appealing and user-friendly cooking companion.
